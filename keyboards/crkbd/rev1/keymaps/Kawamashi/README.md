# Krypton

Le projet Krypton a pour but de fournir une configuration de base fiable et fonctionnelle pour accueillir Propergol et ses spécificités. En effet, Propergol utilise 4 touches de pouce, ce qui impose un certain nombre de contraintes pour la configuration du clavier.

Avec Krypton, je n’ai pas pour ambition de fournir une configuration “parfaite”, mais plutôt une base solide que vous pourrez par la suite modifier pour qu’elle colle au mieux à vos besoins. *Tweak it until you love it !* 


## Propergol et ses contraintes

Propergol utilise déjà 4 touches de pouce, la disposition a besoin d’être installée sur un clavier avec 3 touches par pouce pour fonctionner de manière optimale. Par conséquent, il est compliqué de mettre des touches d’édition (backspace, enter, tab et esc) sous les pouces.
